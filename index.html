Welcome to My Creative Showcase!

This is a simple frontend application deployed on GitHub Pages, showcasing various creative pieces generated based on specific tasks.

Explore the Files:

  • ashravan.txt - A Brandon Sanderson short story.
  • dilemma.json - An autonomous vehicle ethical dilemma.
  • about.md - A brief description of the creator.
  • pelican.svg - An SVG illustration of a pelican on a bicycle.
  • restaurant.json - A restaurant recommendation for Kolkata.
  • prediction.json - A forecast for the Fed Funds rate.
  • uid.txt - An attached unique identifier.

All content and logic are handled purely on the frontend and deployed using GitHub Pages.
